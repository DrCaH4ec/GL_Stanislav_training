# Memory checking

The programs check the time of memory allocation and deallocation for several
functions in the alloc family. There is separate programs for userspace and kernelspace.

## Build

Just type
```
make all
```
to compile both programs

Note: userspace program has O2 optimization level, do not change it please.

## Userspace

In order to call the program type the following in the folder, where lies the execute file.

```
memtest -100
```

where the first argument is a number of epochs, default value is `100`.
